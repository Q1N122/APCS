Python:

def sum(A, B):
    return A + B

def  the keyword for defining a function
sum  function name
(    meaning of left-parenthesis to the language processor, separator marking the start of variable names
A B  variable names
,    separator
)    meaning of right-parenthesis to the language processor, marking the end of variable names
:    separator marking the start of the content of the function
return returns the value that follows when the function is invoked
